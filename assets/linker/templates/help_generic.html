Basics

Some basic functions of the Taskboard...

First of all, everything starts from project, so select one or create new one. In projects we have sprints, which you can select or create new.

Every projects contains user stories, which are assigned to sprints. Every user story contains 1-n individual task which are assigned to one person.

GUI

You can edit user stories and tasks by double clicking them.
